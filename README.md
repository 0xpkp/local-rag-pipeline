# local-rag-pipeline
created full local RAG pipeline, from scratch, without using any external libraries

project outline : 
- source was scraped from multiple different sources
- used cosine similarity to get the best matching source
- used google-gemma-2-1b-it model from huggingface.

outcome :
  this model was accurately able to extract the relevant text from the source and also returned the source from where the relevant text was extracted
